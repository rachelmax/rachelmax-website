Owlfinder

This procedurally generated game was made in Unity using C# scripts. Roam around the low-poly landscapes to find the owl using 3D sound. Once found, the owl will transport you to the next level. There was a bit of an issue getting collison to work on the owl so it remains a work in progress. Download the app here.
