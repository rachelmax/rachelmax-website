LED Umbrella

This LED umbrella was a great project to learn about microcontrollers, color sensors, Arduino code, soldering, circuits, wiring, Lithium Polymer batteries, and amperage. It has a color sensor to match outfits and a rainbow mode. The original design is by Leslie Birch. My version of the umbrella was featured on Adafruit's Wearable Wednesdays. Check out the Umbrella Process page to learn more about how this project was made.
